I'm Zhanna Gablia

Erudite , Purposeful and Sociable.

I develop an interface for accessing lessons on simulators and process the results of their completion. For this purpose, I developed a PHP plugin that integrates into the Moodle LMS via the API of this platform and exchanges data with simulators via HTTP. The plugin supports LTI, which makes it an access point to simulators from other LMSs.
The web interface for accessing simulators as a separate plugin allows you to install it on any Moodle LMS automatically. LTI support gives customers (30 universities) the opportunity to start a lesson on a simulator from their familiar Blackboard LMS.
